B — Split & Time Features

Code: Split 60/20/20

# ── Split theo tỉ lệ 60/20/20 ────────────────────────────────────── n = len(df) train_size = int(n * 0.6) # 41,808 val_size = int(n * 0.2) # 13,936 test_size = int(n * 0.2) # 13,936 train_df = df.iloc[:train_size].copy() val_df = df.iloc[train_size : train_size + val_size].copy() test_df = df.iloc[train_size + val_size :].copy() # ── Thêm 4 time features ────────────────────────────────────────────── def add_time_features(dataframe): idx = dataframe.index t_intra = idx.hour * 4 + idx.minute // 15 dataframe['time_sin'] = np.sin(2 * np.pi * t_intra / 96) dataframe['time_cos'] = np.cos(2 * np.pi * t_intra / 96) dataframe['day_sin'] = np.sin(2 * np.pi * idx.dayofweek / 7) dataframe['day_cos'] = np.cos(2 * np.pi * idx.dayofweek / 7) return dataframe

C — STL Decomposition

⚠️ Chú ý quan trọng: STL được fit chỉ trên tập train để tránh data leakage. Seasonal pattern (1 chu kỳ ngày đại diện – 96 bước) được extract từ train, sau đó tile & apply cho val/test theo đúng vị trí giờ trong ngày thực tế (không bị lệch pha).

Code: STL – fit train, apply val/test

# STEP 1: Fit STL chỉ trên train ─────────────────────────────────── period = 96 # 15-min × 96 = 1 ngày stl = STL(train_df["OT"], period=period) res = stl.fit() train_df["trend"] = res.trend.values train_df["seasonal"] = res.seasonal.values train_df["residual"] = res.resid.values # STEP 2: Extract 1 chu kỳ seasonal đại diện ────────────────────── seasonal_pattern = np.array([ res.seasonal[i::period].mean() for i in range(period) ]) # shape: (96,) # STEP 3: Apply cho val/test ─────────────────────────────────────── for split_df in [val_df, test_df]: split_df["trend"] = apply_trend(split_df, window=period) split_df["seasonal"] = apply_seasonal(split_df, seasonal_pattern) split_df["residual"] = (split_df["OT"] - split_df["trend"] - split_df["seasonal"]).values
❌ Cách sai (Naive)

Fit STL trên toàn bộ df → val/test "thấy" thông tin tương lai → data leakage → metrics giả tạo tốt hơn thực tế.

✅ Cách đúng

Fit STL chỉ trên train, extract seasonal pattern, tile theo phase thực tế cho val/test → không có data leakage.
