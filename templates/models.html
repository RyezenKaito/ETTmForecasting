INPUT x_enc: [B, 336, {{ n_features }}] ↓ ENCODER LSTM (2 layers, hidden=128) → h, c [2, B, 128] ↓ DECODER LSTM (autoregressive, 24 steps) step t: dec_in [B,1,1] → LSTM → pred_t [B,1] (teacher forcing: use y_true or pred_t-1) ↓ OUTPUT: [B, 24] (24 bước dự báo)

Forward pass code

def forward(self, x_enc, y_true=None, tf_ratio=0.0): _, (h, c) = self.encoder(x_enc) # encode dec_input = x_enc[:, -1, self.target_index].unsqueeze(1).unsqueeze(2) predictions = [] for t in range(self.pred_len): dec_output, (h, c) = self.decoder(dec_input, (h, c)) pred = self.fc_out(self.dropout(dec_output[:, -1, :])) predictions.append(pred) # teacher forcing if y_true is not None and random.random() < tf_ratio: dec_input = y_true[:, t].unsqueeze(1).unsqueeze(2) else: dec_input = pred.detach().unsqueeze(1) return torch.cat(predictions, dim=1) # [B, 24]

Temporal Convolutional Network với dilated convolutions – xử lý song song (không autoregressive). Receptive field đủ lớn để bao phủ toàn bộ seq_len=336.

Sơ đồ kiến trúc

INPUT x: [B, 336, {{ n_features }}] → permute → [B, {{ n_features }}, 336] ↓ TEMPORAL BLOCKS ×5 Block 1: in={{ n_features }}, out=32, dilation=1, RF=13 Block 2: in=32, out=64, dilation=2, RF=25 Block 3: in=64, out=128, dilation=4, RF=49 Block 4: in=128, out=128, dilation=8, RF=97 Block 5: in=128, out=256, dilation=16, RF=193 → RF total=373 (each: LayerNorm → Conv1d → GELU → Dropout + residual) ↓ last timestep [B, 256] concat future_cov_proj [B, 64] ↓ FC HEAD: LayerNorm → Linear(320,128) → GELU → Linear(128,24) ↓ OUTPUT: [B, 24]

Receptive Field Calculation

RF = 1 + 2 * (kernel_size - 1) * sum(dilations) = 1 + 2 * (7 - 1) * (1 + 2 + 4 + 8 + 16) = 1 + 12 * 31 = 373 ✓ (≥ seq_len=336)

Attention Mechanism – Scaled Dot-Product

class S2SAttention(nn.Module): def __init__(self, hidden): self.scale = (hidden * 2) ** -0.5 # 1/√(512) def forward(self, h_last, enc_out): # h_last: [B, 512] enc_out: [B, 336, 512] q = h_last.unsqueeze(1) # [B, 1, 512] w = torch.softmax( torch.bmm(q, enc_out.transpose(1,2)) * self.scale, dim=-1) # [B, 1, 336] return torch.bmm(w, enc_out).squeeze(1) # [B, 512]
Ý nghĩa: Ở mỗi decode step, attention weight w[t,s] cho biết model "chú ý" bao nhiêu vào timestep s của lịch sử (336 bước) khi dự báo bước t.
